# Student Management System

## Project Description
This is a full stack web application developed as part of the Full Stack Development Internship at RD INFRO TECHNOLOGY.

## Functional Requirements
- Admin login
- Add student details
- View student list
- Delete student records

## Non-Functional Requirements
- Simple user interface
- Secure authentication
- Fast response time

## Tech Stack
Frontend: React.js  
Backend: Node.js, Express.js  
Database: MongoDB

## Tools Used
- GitHub
- VS Code
- Postman
